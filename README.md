## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/IsmailBinMujeeb/dojo.git
    cd dojo
    ```

2. **Setup Frontend**
    ```bash
    cd client
    npm install
    npm run dev
    ```

3. **Setup Backend**
    ```bash
    cd server
    npm install
    npm run dev
    ```

3. **Open in your browser**
    ```
    http://localhost:5173
    ```
